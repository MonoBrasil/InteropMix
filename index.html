Armazenamento Seguro de Senha

Juliano AtanasioPostgreSQL

17:00

EM BREVE

Quem nunca viu em um site a opção "Esqueci minha senha"? Pois bem. O que acontece quando ali clicamos? Normalmente são duas situações após fornecermos nosso e-mail: 1) É enviado um link temporário para que possamos trocar nossa senha no sistema; 2) A senha é enviada em texto plano para o nosso e-mail. Então vem a pergunta em mente? Qual a forma mais segura de guardarmos senhas de um sistema na base de dados. A palestra mostra de forma simples e objetiva, de forma rápida a teoria sobre o assunto. Como exemplo é demonstrado utilizando a extensão pgcryto que fornece funções para que se faça armazenamento seguro de senhas de uma aplicação dentro da base de dados.
