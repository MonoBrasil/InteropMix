Sobre o Evento

InteropMIX Games, Mobile e Cloud!!!

24 de outubro de 2015, São Paulo e Curitiba

O maior evento de interoperabilidade do Brasil! Venha conhecer as principais tecnologias cross-mobile em C# para plataformas não-Windows com o Time do Projeto Mono Brasil! O Evento é Gratutito para todos este ano!

A Visão

Interoperabilidade entre linguages e sistemas operacionais nunca esteve tão evidente e necessários quanto hoje. Venha conhecer as tecnologias que tem feito difereça e que estão mostrando a muito tempo como será o nosso futuro.

Encontros e Comunidade

Um evento criado pela comunidade do Projeto Mono Brasil para conectar a comunidade Mono e Xamarin aos desenvolvedores que precisam de sistemas interoperáveis.

Venha encontrar-se com os desenvolvedores do Grupo de Usuários Xamarin que vão se reunir nesse evento. Venha conhecer os criadores do Projeto Mono no Brasil,

NOVOS RECURSOS DO VISUAL STUDIO 2015 E C#6.0

Renato Groffe Consultor

Nesta palestra vai ser abordado os novos recursos disponível no Visual Studio 2015 e do C#6.0, vamos abordar também, Using Static, Auto-Porperty Initializers, Expression-bodied Members, Exception Filters, String Interpolation e Null-conditional operator.

10:00

DESENVOLVENDO COM C# E NETDUINO

Claudenir Andrade Gerente P&D - Daruma, Microsoft MVP

Você já deve ter ouvido falar disso. Internet das Coisas (IoT Internet of things), tudo o que é dispositivo pessoal ou equipamento com um endereço IP, sendo monitorado, gerenciado e telemetrado. Mas que tal começar a colocar a mão na massa e entender como interagir com os dispositivos por exemplo Chaves de contato liga e desliga, medidores de potência, leds, tudo isso usando MicroFrameWork + Arduino = NetDuino em C#!!! Nesta palestra você poderá interagir com duas soluções que utilizam Windows Azure como infra para comandar dispositivos, cenário típico de IOT. Utilizaremos uma RaspBerry com Windows IoT, conectada ao Aure a comandando um dispositivo local, e uma caia de controle de chaves On/Off respondendo e sendo comandada por um site de telemetria hospedado no Windows Azure.

11:00

DESENVOLVENDO APPS MULTIPLATAFORMA COM XAMARIN E MVVMCROSS

Alexandre Chohfi Game Developer, Microsoft MVP

Nesta palestra iremos explorar como implementar uma arquitetura MVVM (Model View ViewModel) no Xamarin (Xamarin.Android e Xamarin.iOS) para o desenvolvimento de aplicativos móveis multiplataforma. O Xamarin é um framework de desenvolvimento para Android e iOS e outras plataformas, em linguagem C#. Você vai entender como o Xamarin funciona e como tirar proveito disso para maximizar o reuso do seu código – pensando num ambiente multiplataforma desde a concepção do seu app até a customização de componentes de UI em cada situação.

12:00

COMPARTILHAMENTO DE CÓDIGO ENTRE PLATAFORMAS(WINPHONE, IOS, ANDROID) COM C# E XAMARIN

Ricardo Dorta Microsoft MVP

14:30

Arquitetura para desenvolvimento mobile com Xamarin

Angelo Belchior Arquiteto de Software
