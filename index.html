Sobre o Evento

InteropMIX Games, Mobile e Cloud!!!

24 de outubro de 2015, São Paulo e Curitiba

O maior evento de interoperabilidade do Brasil! Venha conhecer as principais tecnologias cross-mobile, como por exemplo, C# em plataformas não-Windows!

A Visão

Interoperabilidade entre linguages e sistemas operacionais nunca esteve tão evidente e necessários quanto hoje. Venha conhecer as tecnologias que tem feito difereça e que estão mostrando a muito tempo como será o nosso futuro.

Encontros e Caravanas

Venha encontrar-se com os desenvolvedores do Grupo de Usuários Cross-Mobile que vão se reunir nesse evento. Venha conhecer os criadores do Projeto Mono no Brasil, ter contato com palestras das empresa Xamarin, Unity3D, Microsoft entre outras.

Mora em Curitiba e quer vir para o evento? Entre em contato com o Alessandro Binhara, que está organizando uma caravana para vir para São Paulo!

#vamosprogramar - Ensinando a Pensar Desde Já com Android

Francisco Isidro Massetto Universidade Federal do ABC

O projeto #vamosprogramar é um projeto que visa levar uma metodologia para ensino de programação de forma Gamificada. Foi desenvolvido um aplicativo em Android com base no modelo Scratch, porém resumido. O foco deste aplicativo, sendo Android é proporcionar duas principais características: facilidade (o usuário tem o dispositivo à mão rapidamente) e simplicidade (apesar do Scratch ser muito poderoso, ele possui muitos recursos que podem confundir um iniciante). A partir daí uma metodologia baseada em Games e RPG foi desenvolvida, integrando o App a um servidor de correções, que conduz o processo de evolução do estudo e também correções automáticas dos exercícios e desafios propostos. Os scripts criados pelos alunos são corrigidos e um feedback automático é gerado para que cada aluno saiba onde errou e acertou, com base em elementos pré-definidos.

CloudWebIDE + EasyCloud - Um ambiente integrado totalmente na nuvem para desenvolvimento de aplicações

Francisco Isidro Massetto Universidade Federal do ABC

Um dos principais desafios para quem inicia no desenvolvimento de aplicações BigData e Cloud é justamente a falta de ambientes computacionais, bem como a complexidade de configuração inicial destes ambientes. Neste sentido, uma ferramenta que facilite o usuário iniciante a ter contato com as APIs e bibliotecas de programação distribuídas podem facilitar o aprendizado e também poupar boa parte do esforço de configuração. Deste modo, foi desenvolvido na UFABC uma suite de ferramentas compostas pela CloudWebIDE, uma IDE totalmente implementada para executar em um browser que permite aos programadores desenvolverem aplicações Map-Reduce em Java bem como configurar seu ambiente de execução. Nesta segunda funcionalidade, também apresentamos o EasyCloud, uma ferramenta de gerenciamento de máquinas virtuais de forma elástica para proporcionar um ambiente dinâmico de IAAS para os programadores.
